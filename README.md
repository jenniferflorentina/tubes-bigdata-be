### Used:
lang: **go**  
mux: **github.com/gin-gonic/gin**  
mongodb driver: **go.mongodb.org/mongo-driver**  
### How to run app:
First, get libs.
```
go get github.com/gin-gonic/gin
```
```
go get go.mongodb.org/mongo-driver
```
Then run the code.
```
go run main.go
```
It runs on port 8080 by default.
